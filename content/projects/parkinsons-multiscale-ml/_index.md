This project develops a multi-scale analytical framework integrating clinical data with multi-omics data to study Parkinson’s disease.

Key components include:
- Machine learning models applied across heterogeneous datasets  
- Construction of gene regulatory networks for biological interpretation  
- Integration of transcriptomics and metabolomics data  
- Exploration of causal machine learning approaches  

The framework aims to identify robust biomarker signatures and improve interpretability of complex disease mechanisms.
